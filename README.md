# One Click Start
Time-saving tool for every new WordPress project.

One Click Start automates the boring setup tasks every developer repeats — from cleanup to configuration and plugin installation — all in one go.

Perfect for developers, agencies, and freelancers who build multiple websites and want a consistent, reliable starting point.

# Key Highlights

1. Automated Setup: Delete default posts, pages, comments, and bundled plugins like Hello Dolly in seconds.
2. Core Configuration: Set permalinks, disable comments, and adjust comment moderation, search engine visibility, and disable XML-RPC.
3. Bulk Plugin & Theme Installation: Choose from a curated list of popular developer themes and plugins to install automatically.
4. Recipe Export/Import: Save your setup as a JSON recipe and reuse it across projects for consistent workflows.
5. Developer-Friendly: Built with a robust batch handler that avoids timeouts and shows real-time progress.
6. This plugin is lightweight, modern, and designed for professionals who value automation and consistency.

One Click Start lets you focus on building — not setting up basic things again & again.
